# [八一中文网](https://www.zwdu.com/)电子书下载器


抓取小说和封面生成预排版的TXT文件，可手动导入Calibre生成对应的mobi或Epub格式并导入kindle

## 依赖
- python3
- requests
- BeautifulSoup
- pipenv

## 使用方式

1. clone 代码
2. pipenv install
3. python3 main.py
4. 输入八一中文网任意小说目录页面
